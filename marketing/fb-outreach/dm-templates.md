# Facebook Group DM / Outreach Templates

Use these for DMs to admins, replies to "what tools do you use" threads, and warm-intro messages. Lead with value, never with the install link in the first message.

---

## DM 1 — Asking an admin for permission to share

> Hey [name], hope it's a good week. I'm a long-time member of [group name] and I built a free Chrome extension called Deal Scout that scores Facebook Marketplace, eBay, Craigslist, and OfferUp listings using real eBay sold-price data — basically tells you if a listing is a fair deal before you message the seller.
>
> No account, no ads, no paid version, and we don't collect any personal data. I'd love to share it with the group if you're open to it — happy to send you a short demo link first so you can decide.
>
> Either way, thanks for running the group.

---

## DM 2 — Replying to someone who just asked "is this a good price?"

> I had the same question on a [item] last month and built a little tool for it — Deal Scout. It pulls real eBay sold prices and gives the listing a 1–10 score right on the page. Free, no signup. Mind if I DM you the link? Curious what it scores yours at.

---

## DM 3 — Warm follow-up after someone tries it

> Hey — saw you installed Deal Scout, thanks for trying it. Quick favor: if it ever scores something weird (way too high or low), would you screenshot it and send it back? That feedback is how I make it smarter. Either way, hope it saves you a few bucks this week.

---

## Notes
- Always personalize the first line ("saw your post about the [item]")
- Never paste the Chrome Web Store link in the first message — wait for a yes
- If they decline, thank them and move on. Do not follow up twice.
