- Facebook Marketplace
- Craigslist

- A 1–10 deal score with a "GREAT DEAL / FAIR / OVERPRICED / AVOID" verdict

- A copy-paste negotiation message tuned to the data
- Scam and red-flag detection — flags suspicious pricing, missing photos, condition mismatches
- "Buy new" comparison so you know when used isn't worth the hassle
- Auction-aware mode on eBay with a suggested max bid and walk-away price
