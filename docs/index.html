Privacy
Your data stays yours

What Deal Scout does with your data

Deal Scout sends listing content (title, price, description, and photos) to our scoring API to generate your deal score. This data is used only to produce the score and is not stored, sold, or shared.

Deal Scout never reads your browsing history, never tracks which pages you visit, and never collects personally identifiable information.

All API communication uses HTTPS. The extension requests only the minimum permissions required to read listing pages on supported platforms.
